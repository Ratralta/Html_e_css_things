# Principais :
* Sair do shell e voltar ao prompt de comando normal : 
	* **.quit** : Sai do shell/banco de dados. 
	* Também é possível usar o atalho  **"Control + Z"** para sair.
* Entrar em algum banco de dados : 
	* **.open data_base.db** : Entra no banco de dados do nome colocado. 
* Ver as tabelas existentes dentro de algum banco de dadas. 
	* **.tables** : Amostra as tables do banco de dados que você está
