# O que é : 
* O XAMPP ele é um pacote de recursos que ajuda no desenvolvimento web. 
* Um de seus recursos permite criar um [serviodor web](html_servidor_web).
* **De acordo com o chat gpt**, ele é usado somente para testes, não sendo recomendado para uso comercial por não possuir tanta segurança.
