# DOM :

## O que é : 
* **Document Objecet Model** (DOM),  representa todo o  [documento web](html_diferenca_entre_pagina_web_site_servidor_web_e_mecanismo_de_busca) como uma arvore logica para que **linguagens de programação** possam **interpreta-ló**. 
* Ele segue a logica de **"nodes"**, onde o documento web é uma arvore e cada **tag** é uma raiz, as raízes podem serem retornadas como **objetos** através dos métodos do **DOM**.
* Geralmente utilizado no [[JavaScript]], mais esse conceito é do próprio [[HTML]].
