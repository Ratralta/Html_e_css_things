# Pagina Web : 
* Uma pagina web é um **documento/arquivo** que pode ser aberto por um **"navegador web"**, então quando você cria um arquivo de extensão ".html", ele é uma **pagina web**.
* Também é chamado de "pagina"
# Site :
* É um conjunto de **"Paginas Web"** conectadas entre si de diversas formas. 
* Também chamado de "web site".
# Servidor Web :
* É um computador que hospeda um **"site"** na internet, que possui a capacidade de entregar arquivos para o usuário e está conectado a internet. 
* Isso significa que um **hardwere**, guarda os arquivos necessários para fazer uma pagina web funcionar.
* [-- LINK --](https://developer.mozilla.org/pt-BR/docs/Learn_web_development/Howto/Web_mechanics/What_is_a_web_server)
# Mecanismo de busca :
* É um serviço da web que te permite encontrar outras "Paginas Web".
