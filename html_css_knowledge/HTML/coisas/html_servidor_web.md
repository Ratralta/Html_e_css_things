# O QUE É : 
* É um computador que hospeda um **"site"** na internet, que possui a capacidade de entregar arquivos para o usuário e está conectado a internet. 
* Isso significa que um **hardwere**, guarda os arquivos necessários para fazer uma pagina web funcionar.
