# window : 
## O que é : 
* É um objeto global que representa a "janela" do navegador. 
* Nele possui varias funções/variáveis relacionadas a "janela" do navegador . 
* [-- LINK --](https://developer.mozilla.org/en-US/docs/Web/API/Window)
## Funções :
* Abre uma nova janela : 
	* ==window.open(https://www.youtube.com/)== 
* Fecha a janela atual :
	* ==window.close()==
* Gerar um popup na tela.
	* ==window.alert("Cuidado mano")==
* Gerar um popup na tela com input, e retornar o que foi escrito numa variável : 
	* ==variavel = window.prompt("qual seu nome?")==
