# O QUE É :
* NPM --> **Node Package Manager**.
* Ele é um gerenciador de pacotes do "**Node.js**" e um repositório para publicação de projetos Node.js.
* Ele existe pois, para utilizar ferramentas e pacotes do "Node.js", é necessário **instala-los e gerencia-los manualmente**, porem graças ao NPM, ele faz esse trabalho por você através de comandos, facilitando trabalhar com Node.js.
