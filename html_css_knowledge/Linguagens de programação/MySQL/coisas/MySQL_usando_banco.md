# O QUE É :
* É quando você especifica sua [workbank](MySQL_workbank) em um banco de dados especifico. 
* Para fazer isso possui duas maneiras : 
	* Usar o Query "USE" mais o nome do banco de dados que você deseja antes de digitar  seus comandos.
		* EX:
		```MySQL
		USE ronaldo;
		CREATE TABLE nomes(
		leandro VARCHAR()
		);
		```
	* Clique no banco de dados que deseja e clique no botão SQL.
