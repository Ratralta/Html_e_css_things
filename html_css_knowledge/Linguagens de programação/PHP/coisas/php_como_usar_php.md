# Como usar :
* Crie o arquivo de com a extenção ".php". 
* O php precisa de um [servidor web](html_servidor_web) para funcionar, caso você possua o [[PHP]] instalado, use o seguinte comando para abrir um servidor web:
```CMD
	php -S localhost:8000
``` 
* Então em um navegador de internet, coloque "http://localhost:8000/arquivo.php" para entrar no arquivo que deseja manipular e pronto.
* Para fechar o servidor web basta fechar o terminal que você usou este comando.
